# Política de Privacidade - Manga Tracker Sync

Última atualização: 24/02/2026

## Dados coletados
- A extensão pode processar os seguintes dados da página de leitura:
  - Título do mangá
  - Capítulo identificado
  - Identificador externo (slug)
  - Domínio de origem
- A extensão armazena localmente configurações fornecidas pelo usuário:
  - URL da API
  - Slug do parceiro
  - Token de integração
  - Lista de domínios permitidos

## Uso dos dados
- Os dados são usados somente para sincronizar progresso de leitura com a conta do usuário no Manga Tracker.
- A extensão envia dados apenas para a URL de API configurada pelo usuário.

## Compartilhamento
- Não há venda de dados.
- Não há compartilhamento com terceiros fora da infraestrutura do Manga Tracker, exceto quando exigido por lei.

## Retenção e controle
- Configurações ficam no armazenamento da extensão até o usuário remover/alterar.
- O usuário pode desativar a extensão a qualquer momento.
- O usuário pode remover a extensão para apagar dados locais.

## Segurança
- A comunicação com o backend de produção deve usar HTTPS.
- Tokens de integração devem ser tratados como credenciais sensíveis.
